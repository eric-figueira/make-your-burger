# Make Your Burger
A simple website to make your burger. Built in VueJS and following the course [Curso de Vue 3](https://www.youtube.com/watch?v=wsAQQioPIJs&list=PLnDvRpP8BnezDglaAvtWgQXzsOmXUuRHL) by Matheus Battisti - Hora de Codar

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```
